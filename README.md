# NEXUS - USB de Poder Command Center

Portafolio profesional en formato landing + dashboard en vivo.
Construido con HTML5, CSS3, JavaScript Vanilla, JSON y Chart.js.
Sin instalaciones, sin backend, 100% estatico. Consume APIs publicas:
GitHub API, CoinGecko API y Open-Meteo API.

## Como probarlo localmente
python -m http.server 8000
Luego abre http://localhost:8000

## Como publicarlo
1. Sube esta carpeta a un repositorio de GitHub.
2. Ve a Settings > Pages.
3. Selecciona la rama main y carpeta raiz.
4. Tu sitio queda publico en https://TU-USUARIO.github.io/NOMBRE-REPO
